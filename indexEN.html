Description

Bibinary is a game in which you must find the binary code of numbers. It's almost fun.

There is several levels, from A1 to F6. To pass a level, you must find 3 codes. The letter indicates how much help you've got to find the code (cf. below). The number increases with the size of the code you must find.

  • A1, A2, A3, A4, A5, A6 : the number obtained with the current code is diplayed, decimal popups appear, you can swich 0|1 as much as you want.
  • B1, B2, B3, B4, B5, B6 : the number obtained with the current code is no longer diplayed, decimal popups appear, you can swich 0|1 as much as you want.
  • C1, C2, C3, C4, C5, C6 : the number obtained with the current code is no longer diplayed, decimal popups don't appear anymore, you can swich 0|1 as much as you want.
  • D1, D2, D3, D4, D5, D6 : the number obtained with the current code is diplayed, decimal popups appear, if you flip a 0|1 that is not supposed to be flipped, you will have a penality of 10 seconds and a new number will appear.
  • E1, E2, E3, E4, E5, E6 : the number obtained with the current code is no longer diplayed, decimal popups appear, if you flip a 0|1 that is not supposed to be flipped, you will have a penality of 10 seconds and a new number will appear.
  • F1, F2, F3, F4, F5, F6 : the number obtained with the current code is no longer diplayed, decimal popups don't appear anymore, if you flip a 0|1 that is not supposed to be flipped, you will have a penality of 10 seconds and a new number will appear.

There is two modes : the time mode and the infinite mode. In the time mode, you have a limited time (40s) to find as many binary codes as possible. You can start at the level you want (from the lowest level to the highest already reached level). In the infinite mode, you always start at the lowest level and you must go as far as possible. You win when you pass F6. In both modes, the highest reached level appears on the home page.

Personal Data Management

The only stored data is linked to the current configuration (best score, last used mode). It's not used or sent to anyone. It's erased if the application is deleted.
